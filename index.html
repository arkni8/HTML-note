Introduction to HTML

This note starts with introduction to HTML.

HTML is the building block of a website and determines the elements and materials used to make a website.

We start writing HTML out with something we call 'Boilerplate'.

'emmet' is a plugin (or extension) which helps complete the different common types of elements used in HTML. Writing out 'html' gives out the complete boilerplate setup automatically thanks to the plugin emmet.

It is advised to use Emphasis HTML element to italicize elements and Strong HTML element to make text bold.

All of these can be checked along with the associated attributes and syntax on sites like MDN.

Every site has something called Metadata dubbed as 'meta' and defined while putting the template for boilerplate which is accessed by search engines to look up keywords and give more information in general.

We definte 'charset' (Characcter set) so the browser renders the HTML out properly with corresponding encoding. utf-8 covers almost all the symbols from different language from all over the world and generally used for the almost all website by default.

Wondering how to add bullet points in HTML? You do this -

  • 1st item
    • 1st 1st item
    • 2nd 1st item
    • 3rd 1st item
  • 2nd item
  • 3rd item

What did I do? Maybe try inspecting element to find out! *wink wink*

Okay, we added 'unordered list' but how do I add an ordered list( or you may call bullet points)?

  1. Ayyy
    1. 1st item
    2. 2nd item
  2. 3rd item
  3. 4th item

So as you can see, we can put different types of ordered list in html. Just check MDN for all the types of list you can put down.

Have you check how to put hyperlinks in the HTML document? Well, go ahead and inspect the page now and see how a hyperlink is placed.

To organise things and render them in an ordderly fashion in a HTML document, we often use tables. This is how it goes -

1st bakwas 2nd bakwas
heyho heyyo
nayno nonay

Feel free to inspect elements and see how its done.

You can even insert images at table data. That will also work nicely and give you interesting styling option.

Thing to note is attributes to table have been deprecated in favor of CSS styling so attributes like border are no longer used.


Next up, we are going to see how to use HTML forms.

HTML forms are invoked by form keyword from emmet plugin.

You can see how its setup by inspecting element on this page here but here are some of the attributes you might use.

Command type Function
'text' For inputing text
'submit' Works as a submit button.
'textarea' It is a different command, and not used inside input and it is used for inputing a longer phrase.
password To mask the text being input, especially used when typing password.
